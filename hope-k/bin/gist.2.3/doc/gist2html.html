gist2html

    Description: Converts a given Gist file into HTML format.

    Usage: gist2html <file>

    Input:

    • A file that was produced by one of the Gist programs (compute-weights, classify, kernel-pca or project). Note that the program can optionally read from standard input if the <file> parameter is given as a hyphen ("-").

    Output:

    The program prints to standard output an HTMLized version of the given file. This HTML output includes a section that explains how to interpret the results.

    Options: None.

    Calls: none
